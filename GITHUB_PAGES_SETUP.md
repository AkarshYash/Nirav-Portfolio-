## 📋 Next Steps to Deploy

### 1. Enable GitHub Pages

1. Go to your repository: https://github.com/AkarshYash/Nirav-Portfolio-
2. Click **Settings** (top right)
3. Scroll down to **Pages** section (left sidebar)
4. Under **Source**, select:
   - Branch: **main**
   - Folder: **/ (root)**
5. Click **Save**

### 2. Wait 2-3 Minutes

GitHub Pages will build and deploy your site automatically.

### 3. Access Your Live Site

Your portfolio will be live at:
**https://akarshydash.github.io/Nirav-Portfolio-/**

## 🐛 Troubleshooting

### Site not loading?
- Wait 5-10 minutes for first deployment
- Clear browser cache (Ctrl + F5)
- Check GitHub Pages settings

### Animations not working?
- Open in modern browser (Chrome, Firefox, Edge)
- Enable JavaScript
- Check browser console (F12) for errors
